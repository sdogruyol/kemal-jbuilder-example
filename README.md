# kemal-jbuilder-example

An example to use jbuilder.

## Usage

1. Run
```shell
shards install
crystal src/kemal-jbuilder-example.cr
```

2. Open http://0.0.0.0:3000/
Then you can see **`{"hello":"Hello,Jbuilder!"}`**
